# "Car Price Prediction Using Machine Learning"

## Project Overview

This project predicts the selling price of used cars based on features like present price, kilometers driven, fuel type, transmission, and ownership.

It uses Machine Learning techniques to model real-world car price depreciation.

-----------------------------------------------------------------------------------

## Dataset Features

* Car_Name
* Year
* Selling_Price (Target)
* Present_Price
* Driven_kms
* Fuel_Type
* Selling_type
* Transmission
* Owner

-----------------------------------------------------------------------------------

## Technologies Used

* Python 
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

-----------------------------------------------------------------------------------

##  Machine Learning Approach

* Data preprocessing and cleaning
* Feature engineering (Car Age)
* One-hot encoding for categorical features
* Model: Random Forest Regressor
* Evaluation metrics: R² Score, MAE

-----------------------------------------------------------------------------------

## Key Insights

* Car price decreases with age and kilometers driven
* Higher present price increases resale value
* Automatic and diesel cars tend to have higher value
